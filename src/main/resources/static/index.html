⚡ How It Works

Each user gets a bucket of 5 tokens. Every API request consumes 1 token. When the bucket is empty, requests are rejected with HTTP 429. The bucket refills every 10 seconds. To make this work across multiple servers, Redis stores the bucket state with atomic Lua scripts preventing race conditions. Used by companies like Twitter, GitHub, and OpenAI. 🚀
